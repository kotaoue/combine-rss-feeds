# combine-rss-feeds

[![Go](https://github.com/kotaoue/combine-rss-feeds/actions/workflows/test.yml/badge.svg?branch=main)](https://github.com/kotaoue/combine-rss-feeds/actions/workflows/test.yml)
[![GitHub Marketplace](https://img.shields.io/badge/Marketplace-Combine%20RSS%20Feeds-blue?logo=github)](https://github.com/marketplace/actions/combine-rss-feeds)

A GitHub Action that fetches entries from multiple RSS/Atom feeds and combines them into a single RSS 2.0 XML file, sorted by publication date (newest first).

## Features

- Supports both **RSS 2.0** and **Atom** feed formats
- Configurable **per-source entry limit**
- Combined output is **sorted by publication date** (descending)
- Each entry is tagged with the **source hostname** in its title
- Pure Go implementation — **no external dependencies**

## Usage

```yaml
- name: Merge multiple feeds
  uses: kotaoue/combine-rss-feeds@v1
  with:
    feeds: |
      https://zenn.dev/kotaoue/feed
      https://qiita.com/kotaoue/feed
      https://note.com/kotaoue/rss
    limit: 5
    output_file: public/rss.xml
    feed_title: "My Tech Blog Aggregator"
    feed_description: "Aggregated posts from my blogs"
```

## Inputs

| Input | Required | Default | Description |
|---|---|---|---|
| `feeds` | ✅ | — | Newline-separated list of RSS/Atom feed URLs |
| `output_file` | ❌ | `combined_feed.xml` | Path to the generated RSS file |
| `limit` | ❌ | `10` | Max entries per feed source |
| `feed_title` | ❌ | `Combined RSS Feed` | Title for the generated RSS feed |
| `feed_description` | ❌ | `Merged feed generated by combine-rss-feeds action` | Description for the generated RSS feed |

## Full workflow example

```yaml
name: Update RSS feed

on:
  schedule:
    - cron: '0 * * * *'   # every hour
  workflow_dispatch:

jobs:
  combine:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Merge multiple feeds
        uses: kotaoue/combine-rss-feeds@v1
        with:
          feeds: |
            https://zenn.dev/kotaoue/feed
            https://qiita.com/kotaoue/feed
            https://note.com/kotaoue/rss
          limit: 5
          output_file: public/rss.xml
          feed_title: "My Tech Blog Aggregator"

      - name: Commit updated feed
        run: |
          git config user.name "github-actions[bot]"
          git config user.email "github-actions[bot]@users.noreply.github.com"
          git add public/rss.xml
          git diff --cached --quiet || git commit -m "chore: update combined RSS feed"
          git push
```
